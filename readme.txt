《混沌之熵》 模型替换工程
可以使用这个UNITY 工程来进行模型替换，目前可以实现修改主角模型或者某个指定英灵的模型。
可以直接下载工程压缩包，解压后将工程使用UNITY打开。这里使用的是UNITY 2019.4.40版本，其他版本目前的兼容情况不可知。
工程目录下有几个重要的文件夹需要注意：
ModMgrs 用于存放MOD模板的文件夹，其子文件pack我们建议在使用Unity进行资源打包时，将其设置为打包目的地。
ModPackage 这是Steam自带的MDO文件夹，上传MOD必须通过这个工具才能上传，文件夹内部不需要进行任何修改，但是上传MOD必须有这个文件夹。
Testmodel 这里我提供了一个英雄联盟拉克丝的模型资源，您也可自寻模型文件导入Unity，我的视频教程使用的是这个模型。、
如果您要用自己的模型，切记模型是带有骨骼绑定的fbx结尾的文件，切记不要导入obj文件，那个模型是不能动的。
